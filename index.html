Car Visualization System

This interactive D3.js system explores the a1-cars dataset using multiple coordinated visualizations. Use the filters below, then brush over the scatter plot to update the other views.

Scatter Plot – Weight vs MPG

Color represents origin and point size represents horsepower.

Line Chart – Average MPG by Model Year

Shows the trend of average fuel efficiency over time.

Bar Chart – Average MPG by Origin

Compares average fuel efficiency across regions.

Box Plot – MPG Distribution by Origin

Shows distribution, median, and spread of MPG values.
